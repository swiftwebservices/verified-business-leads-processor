# Verified Business Leads Processor Sample

This is a simple Python script that loads and processes a CSV file of verified business leads. It's useful for developers who need to filter, analyze, or integrate leads into their CRM or email campaigns.

## Requirements
- Python 3.x
- pandas: `pip install pandas`

## Usage
1. Install dependencies: `pip install pandas`
2. Place your `business_leads.csv` file in the same directory (with columns like 'email', 'company', 'industry').
3. Run the script: `python script.py`

The script loads the CSV, filters (e.g., by industry), and saves a processed version.
